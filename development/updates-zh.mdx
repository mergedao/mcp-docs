<Update label="2025-02-14" description="发布 Java SDK">
  - 我们很高兴地宣布，由 VMware Tanzu 的 Spring AI 开发的 Java SDK 现在成为了 MCP 的官方 [Java SDK](https://github.com/modelcontextprotocol/java-sdk)。
    这将与我们现有的 Kotlin SDK 一起，扩充我们支持的编程语言列表。
    Spring AI 团队将作为 Model Context Protocol 组织的重要成员维护该 SDK。
    我们非常欢迎他们加入 MCP 社区！
</Update>

<Update label="2025-01-27" description="Python SDK 1.2.1">
  - MCP Python SDK 的 [1.2.1](https://github.com/modelcontextprotocol/python-sdk/releases/tag/v1.2.1) 版本已发布，
    带来了重要的稳定性改进和错误修复。
</Update>
<Update label="2025-01-18" description="SDK 和服务器改进">
  - [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) 中新增简化的类 express API
  - 在[客户端页面](https://modelcontextprotocol.io/clients)新增了 8 个新客户端
</Update>

<Update label="2025-01-03" description="SDK 和服务器改进">
  - [Python SDK](https://github.com/modelcontextprotocol/python-sdk) 中新增 FastMCP API
  - [服务器仓库](https://github.com/modelcontextprotocol/servers)中新增 Docker 化的 MCP 服务器
</Update>

<Update label="2024-12-21" description="发布 Kotlin SDK">

  - 查看[此仓库](https://github.com/modelcontextprotocol/kotlin-sdk/tree/main/samples/kotlin-mcp-server)获取 MCP Kotlin 服务器示例
</Update>
